# Algoritma & Pemrograman Praktik - Teknik Elektro UTY

## Deskripsi

Repositori ini digunakan untuk menyimpan dokumentasi, source code, dan hasil latihan yang dikerjakan selama perkuliahan **Algoritma & Pemrograman Praktik** pada Program Studi **Teknik Elektro**, semester 2, Tahun Ajaran 2025/2026 di Universitas Teknologi Yogyakarta (UTY).

Materi yang terdapat dalam repositori ini mencakup berbagai latihan pemrograman dasar hingga menengah yang bertujuan untuk meningkatkan pemahaman mahasiswa terhadap konsep algoritma, logika pemrograman, serta implementasinya menggunakan bahasa pemrograman yang digunakan dalam perkuliahan.

## Tujuan Repositori

* Menyimpan source code hasil praktikum.
* Mendokumentasikan proses pembelajaran Algoritma & Pemrograman Praktik.
* Menjadi arsip digital untuk tugas dan latihan selama perkuliahan.
* Memudahkan proses pengelolaan dan pengembangan kode program.

## Materi yang Dipelajari

Beberapa topik yang dibahas dalam mata kuliah ini antara lain:

* Dasar-dasar algoritma
* Struktur program
* Variabel dan tipe data
* Operator dan ekspresi
* Percabangan (Selection)
* Perulangan (Looping)
* Fungsi dan prosedur
* Array
* Struktur data sederhana
* Studi kasus dan implementasi program

## Teknologi yang Digunakan

* C++
* Arduino
